# Flat Lighting 💡
Adds a new video setting to make lighting flatter. It does this by removing the shading differences between the sides and top of blocks, which gives the game's lighting a softer or flat appearance.

The effect is similar to the Old Lighting setting from Optifine's internal shaders.

## Comparison 🎨
![A comparison of a village with/wihout the mod.](https://cdn.modrinth.com/data/PXsE1tUn/images/90165a0532b4ddb5f585a108c707c666a80b33fc.gif)

## Usage 👤
After installing the mod, navigate to the game's video settings and set `Cardinal Lighting` to off to make the lighting flatter. Keeping the setting on will keep lighting identical to the vanilla game.
![A screenshot showing the settings menu with Sodium](https://cdn.modrinth.com/data/PXsE1tUn/images/9c83e44a4c15c21d1daa8c9f9c5199d7b6572e79.png)
 
## Dependencies & Compatibility 🔌
This mod has no dependencies. It is compatible with Fabric API, Sodium and Iris.
